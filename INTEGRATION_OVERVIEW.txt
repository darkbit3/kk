================================================================================
                        API ENDPOINTS SUMMARY
================================================================================

1️⃣  POST /api/start-game
    INPUT:  amount, total, cards, randomNumbers, percentage
    OUTPUT: gameId, payout, balance
    STATUS: ✅ WORKING

2️⃣  POST /api/fetch-card
    INPUT:  cardId
    OUTPUT: success/failure message
    STATUS: ✅ WORKING

3️⃣  POST /api/block-card
    INPUT:  id (1-500)
    OUTPUT: success status, remaining cards
    STATUS: ✅ WORKING

4️⃣  GET /api/game-state
    INPUT:  optional gameId
    OUTPUT: game statistics
    STATUS: ✅ WORKING

================================================================================
                       CALCULATION FORMULAS
================================================================================

PAYOUT:
  Payout = (Amount × Total × Percentage) / 100
  Example: (10 × 50 × 20) / 100 = 100

BALANCE:
  Balance = (Amount × Total × (100 - Percentage)) / 100
  Example: (10 × 50 × 80) / 100 = 400
