                           QUICK TEST CODE
═══════════════════════════════════════════════════════════════════════════════

Paste this in browser console:

fetch('/api/start-game', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    amount: 10,
    total: 50,
    cards: [1, 2, 3, 4, 5],
    randomNumbers: [1, 10, 20, 30, 40],
    percentage: 20
  })
})
.then(res => res.json())
.then(data => {
  console.log('Status:', data.status);
  console.log('Game ID:', data.gameId);
  console.log('Payout:', data.payout);     // Should be 100
  console.log('Balance:', data.balance);   // Should be 400
});

═══════════════════════════════════════════════════════════════════════════════
                         KEY FORMULAS USED
═══════════════════════════════════════════════════════════════════════════════

PAYOUT CALCULATION:
  Formula: (amount × total × percentage) / 100
  Example: (10 × 50 × 20) / 100 = 100

BALANCE CALCULATION:
  Formula: (amount × total × (100 - percentage)) / 100
  Example: (10 × 50 × 80) / 100 = 400
